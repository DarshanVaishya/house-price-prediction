House Price Prediction

Get accurate predictions for property prices in Ahmedabad.

Introduction

House Price Prediction is a project created with the aim of help users with purchase of property in Ahmedabad city by giving accurate predictions on the property based on data collected from property listing websites. We have scraped data from website Makaan.com. The process of data cleaning is performed on this scraped data to get rid of unwanted or outlier data to get clean data.

Try it!

Main logic

The main logic of the project is done using machine learning. The clean data is then pre-processed so it is in a format that can be used for machine learning. We have used linear regression model for the project. The processed data is used for the training of the machine learning model. The trained model is available to the users of our website. After the training, the model will be able to predict prices according to the parameters of the house.

Try it!

Our goal

Our main goal is to create an accurate model which can be used by generel population to gain information regarding the property prices in Ahmedabad city. People can freely use this website to research about the prices of property in different areas of the city. Incentives such as this will increase the freedom of people by allowing them to do their own research. We want to create an free and open-source website to enrich the users experience.

Tools and technologies used

All the tools and technologies used in this project are free and open-source which respect the user's privacy. We have secured this website with the latest HTTPS standards as well as encrypted the passwords on the website to provide excellent security to the users.
